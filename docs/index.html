Queryline connects to SQLite, PostgreSQL, MySQL, or SQL Server, learns your schema, and helps analysts explore data in plain English — on desktop or behind your firewall.

  • Connect read-only data sources Import schema metadata and keep write/DDL SQL blocked at runtime.
  • Chat with your data Generate SQL, run queries, inspect results, and render charts from one conversation.
  • Choose your LLM Use local Ollama or cloud providers like OpenAI, Anthropic, and Gemini.
  • Built for teams and individuals Desktop app for analysts, Docker deployment for shared company networks.
